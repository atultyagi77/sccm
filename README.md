# Sanvith Coaching Centre Website

A responsive website for Sanvith Coaching Centre, built with Next.js.

## Features

- Responsive design
- Class information and tutor profiles
- Contact via WhatsApp and Gmail
- Google Maps directions

## Technologies Used

- Next.js
- React
- Tailwind CSS
- Vercel
